Todo Application 📝

A simple Todo List Web Application built using HTML, CSS, and JavaScript. This app allows users to add, view, and delete tasks in an interactive and user-friendly interface.

This project is perfect for beginners learning JavaScript DOM manipulation and basic web development concepts.

Features

Add new tasks to your todo list.

Delete tasks once completed.

Simple and clean interface.

Fully client-side (no backend required).

How to Use

Clone the repository:

git clone https://github.com/surbyanshu2030-cell/todo_application.git


Open index.html in your browser.

Start adding tasks and manage your daily to-do items.
